# News multiplatform app

This is a News app built with Kotlin Multiplatform.
It targets both mobile platforms, iOS and Android.
The UI is shared with Compose Multiplatform.

## Dependencies

Here we list the main dependencies of the project.

* Koin
* Coil
* Ktor
  * Ktor Client Android
  * Ktor Client iOS
* Kotlin Serialization
* KSP (Kotlin Symbol Processing)
* Room
* SQLite bundled
* Kotlin Date Time (Jetbrains)
* Material icons extended (Jetbrains)
